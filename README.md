# 📰 NEWS-HUB — Real-Time News Aggregator Web App

A modern, responsive **News Aggregator Web Application** that fetches and displays the latest headlines using the **NewsAPI**. Built to demonstrate API integration, frontend development, and deployment using GitHub Pages.

---

## 🚀 Live Demo

👉 [https://rohan742006.github.io/NEWS-HUB/](https://rohan742006.github.io/NEWS-HUB/)

> ⚠️ **Important:** This project requires a **NewsAPI key** to function. Without the API key, the app will not load news content.

---

## ✨ Features

* 🗞️ Real-time news updates
* 🌍 Category-based filtering (Technology, Sports, Business, etc.)
* 🔍 Search functionality
* 📱 Fully responsive design
* ⚡ Fast loading & lightweight

## 🔑 How to Get NewsAPI Key

1. Go to: [https://newsapi.org/](https://newsapi.org/)
2. Click **Get API Key**
3. Create a free account
4. Copy your **API Key**

---

## ⚙️ How To Setup Locally

### Step 1: Clone Repository

```bash
git clone https://github.com/rohan742006/NEWS-HUB.git
cd NEWS-HUB
```

### Step 2: Add API Key

Open the JavaScript file where API is used (example: `script.js`) and replace:

```js
const API_KEY = "YOUR_API_KEY_HERE";
```

with:

```js
const API_KEY = "PASTE_YOUR_NEWSAPI_KEY";
```

---

### Step 3: Run Project

Open `index.html` in your browser.

---

## 🔐 Security Note

Do **NOT** expose your API key in public repositories.

For production use:

* Use environment variables
* Use backend proxy server
